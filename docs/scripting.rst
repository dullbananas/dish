Scripting
=========

.. highlight:: none

Dish currently does not really have any scripting features other than comments.


Comments
--------

You can create single-line comments with a ``#``::

   # This is a comment

Or you can create multiline comments using ``#==`` and ``==#``::

   #==
   This is
   a multiline
   comment
   ==#

.. versionadded:: 0.1
   Multiline comments
